##基本使用方式

安装  

	gem install jekyll

进入文件夹 
	
	jekyll new my-site
	cd my-site
	
起服务

	jekyll serve
	
	服务地址：http://localhost:4000

建立

	jekyll build
	
	jekyll build -watch //持续跟踪
